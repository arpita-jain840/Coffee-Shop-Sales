☕** Coffee Shop Sales Analysis**

📌 **Overview**
This project analyzes coffee shop sales data to find trends, customer behavior, and top-performing products using Excel dashboard.

🎯 **Objectives**
Analyze sales by time & day
Identify peak hours
Compare store performance
Find best-selling products

📊 **Key Metrics**
💰 Total Sales: $698K+
👣 Footfall: 149,116
🧾 Avg Bill: $4.69
🛒 Orders/Person: 1.44

🔍 **Key Insights**
⏰ Peak time: 8 AM – 11 AM
📅 Weekdays → consistent sales
📍 High-traffic stores perform better
🏆 Top products: Espresso, Black Tea, Hot Chocolate

🛠️** Tools**
Microsoft Excel (Pivot, Charts, Dashboard)
